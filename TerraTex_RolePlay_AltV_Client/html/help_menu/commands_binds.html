Standard Binds
Taste Funktion
F1 Öffnen des Hilfemenüs
F2 Binds bearbeiten / Hier kannst du auch die meisten der Standardbinds bearbeiten. Aus diesem Grund sind diese hier nicht aufgelistet.
F3 Einstellungen bearbeiten.
Allgemeine Befehle
Command Syntax Alternativen Beschreibung
/payday Zeigt die Berechnung deines letzten PayDays an.
/jailtime Zeigt die restliche Zeit im Knast an.
/cry [text] c Schreit einen Text in den Chat.
/quiet [text] q Flüstert einen Text in den Chat.
/me [text] Schreibt eine Nachricht in den "Tun"-Chat (sendet eine Nachricht ohne :).
/global [text] g Sendet eine Nachricht in den Globalen Chat.
/openchat [text] Öffnet die Eingabezeile des Chattes mit dem vordefinierten Text.
/pay [idOrName] [amount] Gibt einen Spiel den angebenen Betrag vom Handgeld.
/report Öffnet einen Menü mit verschiedenen Meldefunktionen.
Job Befehle
Command Syntax Alternativen Für Job Beschreibung
/getjob Nimmt einen neuen Job an seinem Jobpunkt an.
/quitjob Kündigt einen Job an seinem Jobpunkt.
/startjob Startet den Job.
/jobhelp Zeigt eine Hilfe zu deinen aktuellen Job an.
Fahrzeug Befehle
Command Syntax Alternativen Beschreibung
/seatbelt Legt den Gurt an.
/indicator_left Schaltet den linken Blinker an. - Eine erneute Benutzung deaktiviert diesen wieder.
/indicator_right Schaltet den rechten Blinker an. - Eine erneute Benutzung deaktiviert diesen wieder.
/indicator_warn Schaltet den beide Blinker an (Warnblinkanlage). - Eine erneute Benutzung deaktiviert diesen wieder.
/indicator_off Schaltet alle Blinker aus.
Privat Fahrzeug Befehle
Command Syntax Alternativen Beschreibung
/carlist Zeigt eine Liste aller deiner Fahrzeuge im Chat.
/park Speichert den Parkplatz des Fahrzeugs. An dieser Stelle wird das Fahrzeug nach einen Serverneustart (alle 24h) spawnen.
Fraktions Befehle
Command Syntax Alternativen Nur für Fraktion Beschreibung
/faction [text] f Senden den angegebenen Text in den Fraktionschat.
/news [text] n NEWS Sendet eine Nachricht in den NEWS-Chat.
/su [idOrName] [anzahl] [text] SAPD Gibt den Spieler die angegeben Anzahl Wantedlevel.
/suspects sus SAPD Zeigt eine Liste aller Verbrecher, sortiert nach ihrem Wantedlevel an.
/arrest [idOrName] SAPD Knastet einen Spieler in der Nähe der Zellen ein.
/grab [idOrName] SAPD Zieht einen Spieler ins Auto und fesselt ihn.
/cuff [idOrName] SAPD Fesselt einen Spieler.
/uncuff [idOrName] SAPD Lässt einen gefesselten Spieler frei.
/stvo [idOrName] [anzahl] [text] SAPD Gibt den Spieler die angegeben Anzahl StVO-Punkte. Bei 20 Punkten werden alle Führerscheine automatisch entfernt.
/jailtime [idOrName] SAPD Zeigt die restliche Zeit im Knast des angebenen Spielers an.
Fraktionsleader Befehle (= Rang 6 einer Fraktion)
Command Syntax Alternativen Nur für Fraktion Beschreibung
/invite [idOrNickname] inv Fügt einen Spieler der Fraktion hinzu.
/uninvite [idOrNickname] uninv Entfernt den Spieler aus der Fraktion.
/giverank [idOrNickname] [rangId] frank Setzt einen Spieler auf den angegeben Rang. Rang 1-6, wobei 5 der CoLeader und 6 der Leader ist.
Admin Befehle
Command Syntax Alternativen Ab Rang Beschreibung
/goto [nameOrId] Supporter Teleportiert dich zu dem Spieler
/gethere [nameOrId] Supporter Teleportiert den Spieler zu dir
/spec [nameOrId] Supporter Startet den Spectatormodus
/unspec [nameOrId] Supporter Beendet den Spectatormodus
/ban [nameOrId] [reason] Moderator Bannt einen Spieler vom Server
/tban [nameOrId] [time] [reason] Moderator Bannt einen Spieler vom Server für eine bestimmte Zeit. "time" ist hierbei ein Zeitstring zusammengesetzt aus jeweils einer Zahl und einer Einheit. Beispielsweise würde "1d20h30m40s" für 1 Tag, 20 Stunden, 30 Minuten und 40 Sekunden bannen.
/setfaction [nameOrId] [factionId] [rank] Administrator Setzt ein Spieler in eine Fraktion
/gotojob [jobid] Administrator Teleport zu einem Jobpunkt
/gotoc [pos] [dim=0] Administrator Teleport zu den angegeben Koordinaten Format: x,y,z
/pos Administrator Zeigt die aktuellen Koordinaten an
/veh [hashName] Administrator Spawned ein Fahrzeug
/vehlivery [liveryId] Administrator Ändert die Fahrzeug Variante (livery)
/vehcolors [color1] [color2] Administrator Ändert die Fahrzeug Farbe
/gpm Administrator Gibt den aktuellen Powermultiplikator des Fahrzeugs zurück
/spm Administrator Setzt den Powermultiplikator des Fahrzeugs
/addapart [rentPrice] [name] Serverleitung / Development Erstellt ein neues Apartment
/setWeather [id] sw Serverleitung / Development Ändert das Wetter auf die angegebene ID
/gmx [timeInMinutes] [reason] Serverleitung / Development Startet einen Serverneustart nach x Minuten aus dem angegeben Grund.
/debugclick Serverleitung / Development Startet den Debugmodus des Klicksystems.
/save [info] Development Speichert die Koordinaten in den Logs.
/saveVeh [info] Development Speichert die Fahrzeug Koordinaten in den Logs.
/savem [info] Development Speichert die Koordinaten in den Logs und erstellt einen Marker an der Position.
